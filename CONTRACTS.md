# Contract Glossary

## Core Contracts
- **LendingPool**: Main interaction point for lending/borrowing
- **LendingPoolConfigurator**: Administrative controls
- **LendingPoolAddressesProvider**: Address registry

## Data Providers
- **ProtocolDataProvider**: UI data aggregation
- **PriceOracle**: Asset pricing source
- **LendingRateOracle**: Interest rate data source

## Tokenization
- **aToken**: Yield bearing asset
- **DebtToken**: Base debt tracking
- **StableDebtToken**: Stable rate debt
- **VariableDebtToken**: Variable rate debt

## Libraries
- **ReserveLogic**: Interest math
- **ValidationLogic**: Safety checks
- **WadRayMath**: High precision math
- **PercentageMath**: Percentage calc helpers
